Игры

  • Pixactly

    Онлайн-инструмент для отработки навыка pixel perfect, важного для любого верстальщика. Для того, чтобы проверить свой глазомер, игроку нужно нарисовать прямоугольник по заданным параметрам. Pixactly сравнит его с заданными размерами и оценит саркастично, но справедливо.

  • HexInvaders

    Аркада, которая поможет запомнить HTML-обозначения цветов. Игрок определяет, какой цвет зашифрован в шестизначном коде, и стреляет из пушки по пришельцу соответствующего цвета. Сперва ориентироваться совсем не просто, но чем больше играешь, тем увереннее делаешь выбор!

  • Flexbox Froggy

    Игра для изучения CSS Flexbox. Вместе с лягушонком Фрогги вы научитесь применять flexbox-свойства для позиционирования объектов на странице. ПО мере выполнения инструкций появляются новые персонажи, и каждому нужно с помощью CSS помочь попасть на свою линию

  • Grid Garden

    Это игра по методологии CSS Grid подойдет и взрослому, и ребенку. Уровень за уровнем игрок узнает новые grid-свойства, с помощью которых ухаживает за маленьким морковным садиком. Всего уровней двадцать восемь, и игра помогает понятными инструкциями на каждом этапе

  • Flexbox Defence

    Игра в жанре Tower Defence, которая научит применять Flexbox. Для того, чтобы одолеть врагов, игрок возводит башни - и чем успешнее вы применяете CSS-код, тем крепче и выше ваши оборонительные сооружения. Перед каждым новым уровнем - инструкция по flexbox-свойствам.

  • CSS Diner

    Простая игра, в тайтле которой авторы пишут: «Where we feast on CSS Selectors». Поможет и освоить CSS, и освежить в памяти базовые правила верстки. В CSS Diner игрок с помощью правильной команды — игра помогает инструкциями — должен расположить на столе посуду и продукты. Всего в игре 32 уровня, на каждом из которых задания усложняются. Выполняя их, вы сможете изучить CSS от простых селекторов вроде дивов до first-child и атрибутов input.
